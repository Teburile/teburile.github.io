JS初探

类型

js的类型：

  • Number
  • String
  • Boolean
  • Symbol
  • object
    • Function
    • Array
    • date
    • RegExp（正则表达式）
  • null
  • undefined

Js的数字都是浮点数，没有整型

控制语句

注释、大部分运算符与C++相同

等于为 === (要求类型和值都相等) 不等于为 !==

等于 == 只要求值相等，判断时，类型不同会转化成同种类型然后判断

if-else 语句与C++相同 三目运算符与C++相同

定义函数在返回值的地方书写function即可

function add(num1,num2){
    return num1+num2;
}

while do-while for 循环语句与C++相同

对于迭代类型有 for of for in 两种

for(let i in arr){
    console.log(i);//输出的是角标 0 1 2 3 
}

for(let i of arr){
    console.log(i);//输出的是值 arr[0] arr[1] arr[2]
}

对象

js的对象都是名称-值对，类似于json，值可以是对象，也可以是函数

var obj={
    name:"233",
    isFinished:true,
    child:{
        color:blue,
        size:12
    },
    getname: function(){
        return this.name;
    }
}

obj.child.color;// blue
obj["child"]["size"];//12

js的函数类似于C++的类

function Person(name,age){
    this.name=name;
    this.age=age;
}

var you=new Person("you",20);
//创建一个对象

数组

角标从0开始，有长度函数，长度比最大索引大一，可能a[0] a[1] a[3] 赋值，但a[2]为undefined 则长度为4

遍历数组推荐使用 for..of

函数

js的函数可以传入不确定个数的参数

function add(...args){
    var sum=0;
    for(var i of args){
        sum+=i;
    }
    return sum;
}

为类（对象）添加方法

function personFullName(){
    return this.first+" "+this.last;
}

function Person(first,last){
    this.first=first;
    this.last=last;
    this.fullName=personFullName;
}

function Person(first,last){
    this.first=first;
    this.last=last;
}
var gha=new Person(ha,g);
Person.prototype.fullName=function(){
    return this.first+" "+this.last;
}

网页事件

可以采用匿名函数来实现

document.querySelector('html').onclick=function(){
    alert('2333');//alert弹出弹窗
    var imge = document.querySelector('img');
    let path=prompt("Input the picture's path you want!");//弹出一个输入框
    imge.setAttribute('src', path);//设置元素src属性值为path
    localStorage.setItem('localPath',path);//本地在浏览器中记录值，下次可以直接使用
}
