2021-08-24

CSS初探

CSS一般书写在HTML的头内 由<style></style>包括

CSS规则集：

h1 {color:red;font-size:14px;} /*h1 为选择器，指向对应元素，后面为声明块，一个属性和一个值 */

选择器

选择器用于查找需要设置样式的HTML元素

分为五类：

  • 简单选择器 选择名称 id 类
  • 组合器选择器 依据相互之间的关系（例如<p>下面的<ul>）
  • 伪类选择器 依据特定状态选择
  • 伪元素选择器 选择元素的一部分并设置样式
  • 属性选择器 依据属性值或属性选择
p {
    text-align:center;
    color:red;
}/*元素选择器，含<p>标签的都收到影响*/

#para1 {
    text-align:center;
    color:red;
}/*id选择器 id必须唯一 且不能以数字开头 需要写#来标识是id选择器*/

.crea {
    font-size: 15px;
    color: blue;
}/*类选择器 需要加.来标识是类选择器*，类名不能以数字开头*/

p.crea{
    font-size: 15px;
    color: blue;
}/*复合选择器 可以选择两个条件的交集 HTML的元素可以引用多个类 实现多个类的效果*/

* {
     font-size: 15px;/*15与像素单位之间不能有空格*/
    color: blue;
}/*通用选择器 使用通配符来设置全体元素的样式*/

h1,h2,p,.crea,para1{
     font-size: 15px;
    color: blue;
}/*分组选择器 可以把多个组合并，可以合并类与id*/
/*以上代码为较简单的选择器*/

在HTML中添加CSS

三种方法：

  • 外部

    以外部文件形式存在，需在HTML文件内head内添加link

    <head>
    <link rel="stylesheet" type="text/css" href="src/css/headstyle.css">
    <!-- rel为关系参数 有几个特定值  type由自己定义 描述类型 herf为资源的位置-->
</head>

    其中src/css/headstyle.css应类似于如下（不包含任何html标签）

    h1{
    font-size:15px;
    color:blue;
}
p{
    text-align:center;
}

  • 内部

    即在head元素的style元素内定义

  • 行内

    直接在元素的标签内添加style属性

    <p style="color:blue;font-size:15px;">233</p>

多个样式表冲突时，使用最后定义的样式表

CSS颜色

背景色：background-color

文本色：color

边框色：border:2px solid blue;

可以使用rgb rgba 指定颜色，使用三个十进制数字或者是HEX值（三个16进制数，且以#开头）均可，a是不透明度

可以使用HSL值指定，为色轮上的着色

H:色相 0到360度 为不同颜色

S:饱和度 颜色的强度

L:亮度

HSL可扩展为hsla

背景

  • background-color 背景颜色
  • background-image 背景图像
  • background-repeat 背景重复
  • background-attachment 背景的滚动与否
  • background-position 背景图像的开始位置

边框

可以设置圆角 宽度 样式 颜色等

外边距

可以设置元素与周围其他元素的距离

可以利用百分比、绝对像素、自动、继承父元素等设置

外边距在某些情况下会自动发送合并

允许负值

内边距

会距边距大小自动调节边框大小

不允许负值

宽度、高度

设置的是包含外边距的区域宽度、高度

框模型

即可以将内容包括在其中的一个框

div{
    width:200px;
    padding:10px;
    border:5px solid gray;
    margin:0;
    outline-style:dotted;/*轮廓、此处为点状轮廓*/
} /*盒元素*/

轮廓有宽度、颜色、简写轮廓、轮廓偏移

文本

h1{
    color:#232323;
    background-color:#FFFFFF;
    text-align:left;/*水平对齐*/
    direction:rtl;/*与下一条配合改变文本方向*/
    unicode-bidi:bidi-override;/**/
    vertical-align:top;/*垂直对齐*/
    text-decoration:overline;/*装饰，包含下划线、横穿线、上划线*/
    text-transform:uppercase;/*指定大小写*/
    text-indent:50px;/*首行缩进*/
    letter-spacing: 3px;/*字母间距、可以为负值*/
}

还可以指定行高、词间距、空白处理方式、文字阴影（阴影颜色、模糊）、字体、字体样式（斜体、加粗、变体（变体将小写字母转换为同样大小的大写字母））、字体大小、

添加图标

使用font awesome网站

<head>
    <link rel="stylesheet" href="https://cdn.staticfile.org/font-awesome/4.7.0/css/font-awesome.css">
</head>
<!--引入图标-->
<i class="fa fa-cloud"></i>
<!--在对应的名称前添加fa即可-->
<i class="fa fa-cloud fa-lg"></i>
<!--图标变大-->

链接

/* 未被访问的链接 */
a:link {
  color: red;
}

/* 已被访问的链接 */
a:visited {
  color: green;
}

/* 将鼠标悬停在链接上 */
a:hover {
  color: hotpink;
}

/* 被选择的链接 */
a:active {
  color: blue;
}

可以用文本的装饰删除下划线、可以指定背景色

列表

颜色、样式、迭代标记（数字、黑点还是字母）

表格

边框、表格高度和宽度、水平对齐、垂直对齐、内边距、水平分割线、鼠标悬停效果、斑马纹效果、背景颜色、文本颜色、响应式表格（出现滚动条）
