入门

python的idle(python专用词，python专用的ide)用的是命令行，也就是源文件中的.exe

基本语法

#print用法
print('asda')#输出asda
print("asda")#输出asda
print("asda"+"asd")#+号可以直接连接字符串
"""输出asdaasd,其中双引号单引号通用，可以混用"""
print(1+1)#输出2，可以直接进行四则运算，除法带有小数
print("fh",2)#输出fh 2
#强制类型转换
int('2.5')#输出2，双引号、单引号或不加均可以
float("1.2")#输出1.2
"""python可以直接当成计算器使用，但是字符串不能和数字直接相加"""

循环语句

#while语句，注意冒号和缩进
#缩进可以用空格，也可以用Tab，但不能混用
'''
while condition:
    do
'''
a1=1
while a1<5:
    a1+=1
    print(a1)

#for语句，注意冒号和缩进
'''
for i in queue:
    do
'''
#i是迭代脚标，queue是迭代序列
list=[6,3,3,4,5,6]
for i in list:
    print(i)

上面两种循环语句均支持continue 和 break操作

外部模块（头文件）

外部模块安装方式：

  • anaconda平台安装
  • 直接下载安装包手动安装：
    • 在包的官网下载对应python版本的whl包
    • 然后使用pip（python包管理工具）安装

pip使用

pip install [name] #直接安装包
pip install 'Markdown>2.0,<2.0.3 #指定版本安装
pip uninstall [name]
pip list
pip freeze > a.txt #将已经安装的库列表保存到文本文件中
pip install -r a.txt #根据依赖文件批量安装库
pip install a.whl #离线安装whl后缀的包
pip install --upgrade [name] #更新包
pip show --files [name] #查看包
pip list --outdated #查看可更新的包
pip源的选择

很多时候，比如网络不给力，连接超时、防火墙阻挡等等各种原因，可能无法从Python官方的PyPi仓库进行pip安装，这时候可以选择国内的第三方源，推荐使用豆瓣源，速度不错。

使用方法：

pip install -i https://pypi.doubanio.com/simple/ --trusted-host pypi.doubanio.com pillow

pip install -i https://pypi.tuna.tsinghua.edu.cn/simple

需要注意的是，除了最后的pillow用所期望的库名替代外，前面的参数都是固定写法，包括参数顺序。
