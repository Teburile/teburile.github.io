爬虫

  1. 引入专用库

    from urllib.request import urlopen
from bs4 import BeautifulSoup

  2. 调用urlopen获得一个网页

    html1=urlopen("https://morvanzhou.github.io/static/scraping/basic-structure.html").read().decode('utf-8')#中文要求转换

  3. 使用BeautifulSoup解析

    soup=BeautifulSoup(html1,features='lxml')
#其他指令
soup.h1#h1标题
href=soup.find_all('a')#找到所有的a标签,生成一个dict的list
print(herf[1]['herf'])#直接通过下标访问

  4. 爬取css代码，其中的类可能放在head中，通过类名定位想要的内容

    jan=soup.find('ul',{"class":'jan'})#查找所有ul标签下类为jan的东西，并生成list
#找到<ul class="jan">
#<li>一月一号</li>
#<li>一月二号</li>
#<li>一月三号</li>
#</ul>

    即每次可以根据标签生成一个list，然后可以用类来特化标签再生成，之后可以用.get_text()得到标签所含的文本，其中的类名可以用正则表达式来匹配

    img=soup.find_all("img",{"src":re.compile('表达式')})
